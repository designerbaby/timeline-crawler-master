# timline-crawler

timeline服务中需要用到的爬虫程序

## 启动

本地: npm run dev 或者 node index.js

## 爬取地址：

[https://netblocks.org/reports](https://netblocks.org/reports)

## 使用到的技术

 - nodejs
 - cheerio
 - moment
 - node-cron
 - node-xlsx
 - request
